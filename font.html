Typography System

The single source of truth for type on the site. Tokens live in styles.css :root; this page mirrors them as a visual specimen. Two fonts carry the whole site — Playfair Display for editorial headings, Inter for everything else. Material Symbols is icon-only.

Implementation notes

Two deliberate deviations from the original token spec, made to honour "don't change layout/spacing".

--font-icon is Material Symbols Outlined, not "Rounded" — the site already loads Outlined everywhere; switching families would change every icon glyph.
No global p { margin-bottom } / ul,ol { padding-left } / li { margin-bottom }. This site spaces paragraphs and lists with flex gap and explicit margins; adding global block margins would shift many existing layouts. Lists keep list-style: none. Revisit when rolling the system out to the remaining pages (Phase 2).
Phase 1 — homepage + shared components (header, nav, footer, buttons, cards, forms, global sections). Legacy aliases --font-serif / --font-sans still resolve to --font-display / --font-body so unmigrated pages keep working. index_old.html is excluded.
